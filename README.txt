最后一次更新：
	1.取消了图片的LZW算法压缩/解压功能

图形界面部分：
	1.平移图片
	2.带负反馈修正的图像缩放
	3.自动调整的最大缩放倍数（保证内存占用合理、软件运行流畅）
	4.“适应窗口”模式，展示图片全貌（大图与小图策略不同）
	5.“适应内容”模式，保证显示质量

软件功能部分：
	1.支持打开BMP,JPG,JPEG,PNG格式的图片文件
	2.支持将编辑后的图片另存为BMP,JPG,JPEG,PNG格式
	3.支持水平翻转、垂直翻转处理
	4.支持灰度处理、反色处理
	5.支持4-邻域平滑、8-邻域平滑
	6.支持将图片转为TXT格式的艺术化处理
	7.支持亮度调节、对比度调节、二值处理（黑白处理）
	8.支持在编辑中随时恢复原图
	9.支持将24位真彩色图片转换为8位265色图片(流行色算法)
其他：
	1.软件退出后自动清除临时文件
